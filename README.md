# Node Authentication App

## Project clone
```
git clone https://github.com/anil-bomma/node-passport-jwt
```

### Install project and dev dependencies
```
npm install
npm install --dev
```

### Compiles and minifies for production
```
npm run dev
```

### Lints and fixes files
```
npm run lint
```
